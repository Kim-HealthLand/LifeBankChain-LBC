# LIFEBANK CHAIN Project Documentation

This directory centralizes project design materials, including architecture design, database design, system component design, and API module design.

## Directory Structure

- `architecture/`: Architecture design, including overall architecture diagrams, flow diagrams, and deployment views.
- `database/`: Database design, including data models, table structures, indexes, and relationships.
- `components/`: System component design, including core modules, service boundaries, and component responsibilities.
- `api/`: API module design, including endpoint lists, request and response structures, authentication, and error codes.
- `smart-contracts/`: Smart contract architecture, token standards, and on-chain authorization design.
- `security/`: Security, privacy, admin permission, consent, and audit design.
